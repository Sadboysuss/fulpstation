/tg/ Station 13 - Modules - Types

fulp_modules/features/mentors/mentorwho.dm

/proc/get_list_of_mentors Proc that returns a list of cliented mentors. Remember that this list can contain nulls! Also, will return null if we don't have any mentors.
/proc/get_general_mentorwho_information Proc that gathers mentorwho information for a general player, which will only give information if an admin isn't AFK, and handles potential fakekeying. Will return a list of strings.
/proc/get_sensitive_mentorwho_information Proc that gathers mentorwho information for mentors, which will contain information on if the admin is AFK, readied to join, etc. Only arg is a list of clients to use. Will return a list of strings.
